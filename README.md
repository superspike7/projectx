# ProjectX

REST API fetching Using CoinGecko

## Screenshots

![App Screenshot](https://via.placeholder.com/468x300?text=App+Screenshot+Here)


## Tech Stack

**Client:** React, Vite, TailwindCSS, React-Query

**API:** CoinGecko


## Installation

Clone and install ProjectX with yarn

```bash
  git clone 
  cd projectx 
  yarn install
```
    
## Local Development

```bash
  yarn dev
```
