# ProjectX

REST API fetching Using CoinGecko

## Tech Stack

**Client:** React, Vite, TailwindCSS, React-Query

**API:** CoinGecko


## Installation

Clone and install ProjectX with yarn

```bash
  git clone https://github.com/superspike7/projectx.git
  cd projectx 
  yarn install
```
    
## Local Development

```bash
  yarn dev
```
